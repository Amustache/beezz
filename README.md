# Beezz

## SAVE THE QUEEN

As the kind and generous bee that you are, go help your sick queen and your  sisters  by collecting pollen in the garden

## COMMANDS

Press C to Start the Game and Start/End Dialogues

Arrows to Move

Press X to Interact
